## 📦 扩展版 UAVSwarmV2 数据集

UAVSwarmV2 数据集在初代 UAVSwarm数据集 基础上进行了全面升级，针对**低空无人机集群目标检测与跟踪**任务优化，适用于多目标跟踪（MOT）、小目标检测等研究方向。
UAVSwarmV2 dataset has undergone a comprehensive upgrade based on the first generation of UAVSwarm Dataset, optimized for the task of **low-altitude drone swarm target detection and tracking**, and is suitable for research directions such as multiple object tracking (MOT) and small target detection.
